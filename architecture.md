# ARQUITETURA DE LABIRINTO (Navegação USDD)

"A linha reta é para os fracos; a espiral é para os filósofos."

### 1. O Loop de Möbius
Toda jornada do usuário deve conter ao menos um ciclo infinito onde ele retorna ao ponto de partida sem explicação clara. Isso promove a reflexão sobre o caminho percorrido.

### 2. Breadcrumbs Alucinógenos
Os caminhos de navegação (ex: Home > Conta > Configurações) devem gerar links aleatórios. Se o usuário clicar em "Home", ele deve ser levado para "Termos de Uso" de 2012.

### 3. Z-Index de Schrödinger
Componentes críticos (como modais de erro) devem ter seu Z-index alternado dinamicamente. Eles estão simultaneamente à frente e atrás do conteúdo, tornando o clique uma questão de sorte quântica.

### 4. Menu Almôndega (Meatball Menu)
Diferente do menu hambúrguer, o menu almôndega é um círculo perfeito de 5px que, ao ser clicado, foge do cursor (utilizar `position: absolute` e `Math.random()`).
